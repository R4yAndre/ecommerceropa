Catálogo de productos: 

1. Declarar variables para almacenar el nombre y precio de cinco productos.

2. Declarar una variable para ingresar el nombre del producto a buscar.

3. Declarar una variable lógica para indicar si el producto fue encontrado (inicialmente es falso).

4. Asignar los nombres y precios a los productos (en este caso, 5).

5. Mostrar en pantalla el catálogo completo con nombres y precios.

6. Pedir al usuario que ingrese el nombre de un producto que desea buscar.

7. Comparar el nombre ingresado con cada uno de los cinco productos (ignorando mayúsculas o minúsculas).

8. Si se encuentra coincidencia:

    8.1 Mostrar el producto y su precio.

    8.2 Cambiar la variable encontrado a verdadero.

9. Si no se encuentra ninguna coincidencia después de revisar todos los productos:

    9.2 Mostrar mensaje de "Producto no encontrado".







Carrito de compras: 

1. Declarar variables para los nombres y precios los productos (5 en este caso).

2. Declarar variables para llevar el conteo de cuántas unidades se agregan al carrito de cada producto (inicialmente en 0).

3. Declarar variables para manejar la selección del producto, la cantidad a agregar y el total a pagar.

4. Asignar nombres y precios a cada uno de los productos.

5. Iniciar las cantidades de cada producto en cero.

6. Mostrar repetidamente el catálogo de productos, dando la opción al usuario de elegir un producto por número o finalizar la compra con la opción 0.

7. Si el usuario selecciona un producto válido:

    7.1 Pedir cuántas unidades desea agregar.

    7.2 Sumar esa cantidad al conteo del producto correspondiente.

8. Repetir el proceso hasta que el usuario elija la opción 0.

9. Calcular el total multiplicando la cantidad de cada producto por su precio y sumando los resultados.

10. Mostrar un resumen del carrito indicando los productos agregados, sus cantidades, subtotales y el total a pagar.





Registro e inicio de sesión:

1. Declarar variables para guardar el nombre de usuario y la contraseña del registro.

2. Declarar variables para ingresar el usuario y la contraseña que se usen al intentar iniciar sesión.

3. Declarar una variable para controlar la opción del menú y otra para saber si la sesión fue iniciada (inicialmente en falso).

4. Solicitar al usuario crear un nombre de usuario y una contraseña para registrarse, y guardar esos datos.

5. Mostrar un mensaje de confirmación del registro.

6. Mostrar un menú que permita:

    Opción 1: intentar iniciar sesión,

    Opción 2: salir del programa.

7. Si el usuario elige iniciar sesión:

    7.1 Solicitar el nombre de usuario y la contraseña.

    7.2 Comparar ambos valores con los datos registrados.

    7.3 Si coinciden, mostrar un mensaje de bienvenida y marcar que la sesión ha sido iniciada.

    7.4 Si no coinciden, mostrar un mensaje de error.

8. Repetir el menú mientras el usuario no haya iniciado sesión ni elegido salir.

9. Al salir del ciclo, si la sesión no fue iniciada, mostrar un mensaje indicando que no se logró iniciar sesión.





Panel administrativo:

1. Declarar variables para almacenar los nombres y precios de cinco productos predefinidos, más un nombre y precio nuevo.

2. Declarar una variable para saber cuántos productos hay actualmente (inicialmente 5).

3. Mostrar los cinco productos con su precio al inicio de cada ciclo.

4. Mostrar un menú con tres opciones:

    Agregar un nuevo producto.

    Reemplazar un producto existente.

    Salir del programa.

5. Si elige agregar producto:

    5.1 Pedir el nombre y precio del nuevo producto.

    5.2 Aumentar en uno el conteo de productos.

    5.3 Asignar el nuevo nombre y precio a la primera posición vacía entre las cinco.

    5.4 Mostrar mensaje de confirmación.

6. Si elige reemplazar producto:

    6.1 Pedir el número del producto que se quiere reemplazar (entre 1 y 5).

    6.2 Si es válido, solicitar el nuevo nombre y precio.

    6.3 Reemplazar el producto correspondiente.

    6.4 Mostrar mensaje de confirmación.

    6.5 Si no es válido, avisar con un mensaje de error.

7. Repetir el menú hasta que el usuario elija la opción de salir.

8. Mostrar mensaje de salida del panel administrativo.
